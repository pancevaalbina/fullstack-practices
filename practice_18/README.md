# Контрольная работа №3

Результат выполнения практических работ № 13-18 по дисциплине "Фронтенд и бэкенд разработка".
Приложение реализует функционал управления заметками с поддержкой офлайн-доступа (Service Worker, Web App Manifest), мгновенной загрузки (App Shell), реального времени (WebSocket) и напоминаний через Push-уведомления (web-push).

## Запуск
Для тестирования функционала необходимо использовать файлы из папки `practice_17`. 
1. Выполните команду `npm install` для установки зависимостей сервера.
2. Запустите сервер: `node server.js`
3. Запустите клиент по HTTPS с помощью `http-server` (используя ключи `mkcert`).
